# 191126 객체

## Call by value

`content` 값이 복사되어 다른 주소에 새로 만들어진다.

## Call by pointer

`주소` 값이 복사되어 같은 다른 주소에 새로 만들어진다. 이 때의 content는 **주소값**이다.

## Call by reference

**`자료형 + &`** 형식으로 사용할 수 있으며 실제 데이터를 가리키는 **또다른 별칭**이다.

```c++
void swap(int &x, int &y){
    int temp;
    temp = x;
    x = y;
    y = temp;
}
```
