### DDL

- Data Definition Language (데이터 정의어)
- 스키마에 대한 정의
- 종류
  - CREATE
  - DROP
  - ALTER
  - RENAME



### DML

- Data Manipulation Language (데이터 조작어)
- 데이터에 대한 조작
- 종류
  - SELECT
  - INSERT
  - UPDATE
  - DELETE

### 제약조건

- cascade : 참조하는 튜플이나 값을 연쇄적으로 삭제 / 변경
  - on delete : 참조하는 모든 튜플들을 삭제
  - on update : 참조하는 모든 Rental 튜플의 accountId를 새로운 값으로 변경
- set null
- set default
- not null
