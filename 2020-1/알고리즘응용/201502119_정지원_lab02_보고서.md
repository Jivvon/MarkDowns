### Goal2

##### Manhattan ($l_1$- distance)

```python
def manhattan(data):
    l = len(data)
    ret = np.zeros((l,l), np.float_)
    for i in range(l):
        for j in range(i+1,l):
            ret[i][j] = ret[j][i] = sum(abs(data[i] - data[j]))
    return ret
```

$\Sigma_{i=1}^n{|x_i-x_j|}$을 코드로 표현하였다.

Manhattan matrix의 대각성분은 0이므로 0으로 초기화시켰다.

symmetric matrix이므로 비교되는 데이터 기준으로 뒤의 데이터와 계산하였다.



##### Euclidean ($l_2$- distance)

```python
def euclidean(data):
    l = len(data)
    ret = np.zeros((l,l), np.float_)
    for i in range(l):
        for j in range(i+1,l):
            ret[i][j] = ret[j][i] = np.sqrt(sum((data[i] - data[j]) ** 2))
    return ret
```
$(\sqrt{\Sigma_{i=1}^n{(x_i-x_j)^2})}$ 를 코드로 표현하였다.

Euclidean matrix의 대각성분은 0이므로 0으로 초기화시켰다.

symmetric matrix이므로 비교되는 데이터 기준으로 뒤의 데이터와 계산하였다.
