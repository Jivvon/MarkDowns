## Classfication

#### Supervised Learning (문제지 방식)

적어도 훈련 데이터 안에서는 100점을 맞을 수 있도록.

1. 예제 문제를 풀게 하고
2. 정답과 비교하여
3. 잘 맞추는 방향으로 반복해서 학습

#### Classification (분류)

기본적인 전제 조건으로는 Class가 정의되어야 한다.

1. 정해진 Class에 Score를 부여
2. 가장 높은 Scoring을 가진 Class 선택

###### Classification @ Deep Learning | Design time

1. N 개 Class 정의
2. Network 구성
3. 입력
4. 계산
5. Class별 Scoring
