# Probability 확률

##### 양궁 선수가 9점 8점 10점을 순서대로 맞췄다. 다음번에 10점을 쏠 **가능성**은?

#### $$P(A) = \frac {n(A)} {n(S)}$$

- 사전확률 Prior probability : Data에 기반하지 않은 확률

- 사후확률

##### 복수의 Event를 고려해보자 !

- Event B가 발생한 후 Event A가 발생할 가능성 : $P(A|B)$ A는 관심사건. B는 일어난 사건.

- #### $P(A|B) = \frac {P(A∩B)} {P(B)}$

- #### 독립 사건 A와 B : $P(A ∩ B) = P(A) * P(B)$



`확률분포를 구하기 위해서 항상 모든 사건을 경험해야 하나 ?`

> No !
>
> **특정 종류**의 Data(=Event 집합)의 경우 **확률계산 방법을 패턴화** 시킬 수 있음



##### 3점 슛을 0.2의 확률로 넣는 농구선수가 몇 번을 던져야 최초로 성공할까 ?

- 기하분포 (Geometric Distribution)
  - 일련의 **독립시행** 실시
  - 각 시행은 **성공 아니면 실패**이며, **성공할 확률은 모두 동일**
  - **첫 번째 성공을 거두기 위해 시도해야 하는 시행의 횟수**가 관심사일 때
  - $P(X = r) = p * (1 - p)^{r-1}$ : r 번만에 성공할 확률
  - $X$ ~ $Geo(p)$ : 사건 X는 Geometric Distribution을 따른다

#### 대표적인 확률 분포

- Discrete Distributions 이산 확률 분포
  - Binomial distribution 이항 분포
  - Bernoulli distribution 베르누이 분포
  - Poisson distribution 포아송 분포
  - Geometric distribution 기하 분포
- Continuous Distributions 연속 확률 분포
  - **Normal distribution (=Gaussian Distribution)** 정규분포 (가우스 분포)
  - Uniform distribution 유니폼 분포
  - Gamma distribution 감마 분포
  - Student's t-distribution 스튜던트 t-분포
