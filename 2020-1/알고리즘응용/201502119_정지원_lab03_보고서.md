좌측부터 순서대로 일어났으며, (0.1,0.2) 부근의 주황색과 파랑색 데이터와 (0.5,0.2) 부근의 보라색과 연두색 데이터를 보면 군집별 데이터가 달라진 것을 볼 수 있다.

```python
# 모든 데이터를 가장 가까운 센터값 그룹에 매핑
def clustering(self, cluster):
    # 각 센터값의 cluster에 속한 data 초기화
    for key, value in cluster.items():
        if len(value['data']) > 0:
            value['data'].clear()
    # 각 데이터마다 모든 센터값을 보고 가장 가까운 곳에 매핑
    for point in self.data:
        tmp = []
        for key, value in cluster.items():
            tmp.append(np.sqrt(sum((point - value['center']) ** 2))) # euclidean
        center_idx = np.argmin(tmp)
        cluster[center_idx]['data'].append(point)
    return cluster
```
