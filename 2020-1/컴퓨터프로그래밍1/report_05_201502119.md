###   4.2 조건문(if문, switch문)

- **if문**
  - 조건식이 true이면 블록을 실행하고 false이면 블록을 실행하지 않는다.
```java
if(조건식){
	실행문;
}
--- 혹은 ---
if(조건식)
	실행문;
```
  - **if-else문**
      - 조건식을 만족하지 않는 경우 else 블록을 실행한다. (복수(병렬)로 가능, 중첩도 가능)
```java
if(조건식){
	실행문;
}else {
	실행문;
}
--- 혹은 ---
if(조건식1){
	실행문;
}else if(조건식2){
	실행문;
}else {...}
--- 혹은 ---
if(조건식1){
	실행문;
  if(조건식2){
		실행문;
  }else{
		실행문;
  }
}
```
- **switch문**

  - 변수나 연산식의 값에 따라 실행문을 선택할 수 있다.
  - break를 만나면 switch문을 나간다.
  - default는 어떠한 맞는 값도 찾지 못했을 시 실행된다.
```java
switch(변수){
	case 값1:
		실행문;
		break;
	case 값2:
		실행문;
		break;
	default:
		실행문;
		break;
}
```
