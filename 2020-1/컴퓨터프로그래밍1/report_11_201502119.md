# chapter07 상속

##### 7.1 상속이란?

- 부모와 자식 관계.
- 자식(하위, 파생) 클래스가 부모(상위) 클래스의 멤버를 물려받는 것
  - 상속 대상: 부모의 필드와 메소드
- 상속 대상 제한
  - 부모 클래스의 private 접근자를 가진 필드와 메소드는 접근 불가능.
  - 부모 클래스가 다른 패키지에 있을 경우, default 접근 갖는 필드와 메소드도 접근 불가능.

##### 7.2 클래스 상속(extends)

- extends는 자식이 상속할 부모를 지정하는 키워드로, 자바에서는 단일 상속만을 지원한다.

##### 7.3 부모 생성자 호출(super)

- **부모 없는 자식은 없다** : 자식 객체 생성할 때에는 부모 객체를 먼저 생성한 후 자식 객체가 생성된다.
- 부모 객체를 생성할 때, 명시적으로 super(매개변수)를 통해서 부모 생성자를 호출할 수 있다.
  - 반드시 자식 생성자의 첫 줄에 (가장 위에) 위치해야하며, 부모 생성자가 없다면 컴파일 오류가 발생한다.

##### 7.4 메소드 재정의(@Override)

- 메소드 재정의란, 부모 클래스의 상속 메소드를 수정하여 자식 클래스에서 재정의 하는 것이다.
  - 접근 제한을 더 강하게 오버라이딩 불가
  - 새로운 예외 throws 불가
- @Override 어노테이션
  - 컴파일러에게 부모 클래스의 메소드 선언부와 동일한지 검사 지시 -> 웬만하면 붙여주자
- 자식 클래스에서 super를 사용하면 부모 객체를 참조할 수 있다.

##### 7.5 final 클래스와 final 메소드

- final 키워드는 **수정 불가능**
- final로 부모 클래스에서 메소드를 생성하면 자식이 재정의하지 못한다.

##### 7.6 protected 접근 제한다

- 같은 패키지 : default와 동일
- 다른 패키지 : **자식 클래스만 접근 허용**

##### UML

<img src="/Users/jiwon/Library/Application Support/typora-user-images/스크린샷 2020-05-31 오후 11.01.14.png" alt="스크린샷 2020-05-31 오후 11.01.14" style="zoom:80%;" />

Circle 클래스가 부모, Cylinder 클래스가 자식인 상속관계를 가지고 있다.

Circle 클래스의 필드로는 double형의 radius 변수와 String 형의 color 변수가 접근제한자 private으로 정의되어있다. 생성자는 기본 생성자와 radius를 받는 생성자 두 개가 있고, radius를 반환하는 getRadius(), area(넓이)를 반환하는 getArea()가 접근제한자 public으로 정의되어있다.

Cylinder 클래스의 필드로는 double형의 height 변수가 접근제한자 private으로 정의되어있다. 생성자는 기본 생성자와 radius를 받는 생성자, radius와 height를 받는 생성자 세 개가 있고, height를 반환하는 getHeight(), volume(부피)를 반환하는 getVolume()이 접근제한자 public으로 정의되어있다.
