# chapter03 연산자

### 3.1 연산자와 연산식

	x+y (`+`는 연산자, 'x'와 'y'는 피연산자)
연산자는 필요로 하는 피연산자의 수에 따라 단항, 이항, 삼항 연산자로 구분된다. 

> 단항 연산자 : ++x : 피연산자가 단 하나뿐인 연산자를 말한다.
> 이항 연산자 : x+y : 피연산자가 두 개인 연산자를 말한다.
> 삼항 연산자 : (sum  > 90) ? "A" : "B"

연산식은 다른 연산자의 피연산자 위치에도 올 수 있다.

    boolean result = (x+y) < 5;

###   3.2 연산의 방향과 우선순위

1. 단항 > 이항 > 삼항 연산자
2. 산술 > 비교 > 논리 > 대입 연산자
3. 단항과 대입 연산자를 제외한 모든 연산의 방향은 왼쪽에서 오른쪽이다.($\rightarrow$)
4. 복잡한 연산식에서는 `괄호`를 사용해서 우선순위를 정해준다.

### 3.3 단항 연산자

- **부호 연산자(+,-)** : 양수 및 음수를 표현하지만, 변수 값의 부호를 유지하거나 바꾸기 위해서도 사용된다. + 연산자는 변수 값의 부호를 유지한다. - 연산자는 변수 값이ㅡ 부호를 양수는 음수로, 음수는 양수로 바꾼다. 반환형은 int 타입.
- **증감 연산자(++,--)** : 변수의 값을 1 증가(++) 시키거나 1 감소(--) 시킨다. 
> ++x	다른 연산을 수행하기 전에 x값을 1 증가
--x	다른 연산을 수행하기 전에 x값을 1 감소
x++	다른 연산을 수행한 후에 x값을 1 증가
x--	다른 연산을 수행한 후에 x값을 1 감소
- **논리 부정 연산자(!)** : boolean 타입에서만 사용하며 true와 false만 값으로 가질 수 있다.
- **비트 반전 연산자(~)** : 정수 타입(byte, short, int, long)의 피연산자에만 사용되며, 피안산자를 2진수로 표현했을 때 비트값 인 0을 1로, 1은 0으로 반전한다. 연산 후, 부호 비트인 최상위 비트를 포함해서 모든 비트가 반전되기 때문에 부호가 반대인 새로운 값이 산출된다. 반환형은 int 타입.

### 3.4 이항 연산자

- **산술 연산자** : + , - , * , / (몫) , % (나머지)
- **대입 또는 복합대입연산자** : =,+=,-=,*=,/=,%=,&=,^=,|=,<<=,>>=,>>>=
- **비교 연산자** : > , < , == , !=
- **논리 연산자** : && (and) , || (or) , ! (not)
- **비트 연산자** : &,|,^,~,<<,>>,>>>
- **쉬프트 연산자** : << , >>
- **문자열 연산자**
- 기본 타입인 변수의 값을 비교할 때와 다르게, 참조 타입인 String 변수를 비교할 때는 equals()를 사용한다. 

### 3.5 삼항 연산자

```
[조건] ? [참일 경우 실행] : [거짓일 경우 실행]
```
